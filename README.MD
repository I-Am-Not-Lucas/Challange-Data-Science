## Explicação
O challange de dados é um desafio promovido pela alura de duração de 4 semanas, com intuito de praticar uma gama de skills para formação de um cientista de dados

## Proposta do desafio
Você é a pessoa contratada como cientista de dados pela operadora de telecomunicações Novexus. 
Na reunião inicial com as pessoas responsáveis pela área de vendas da empresa, 
foi explicada a importância de reduzir a Taxa de Evasão de Clientes, também conhecida como Churn Rate.

## Etapas

:arrow_forward: Semana 1: ETL e EDA  

:arrow_forward: Semana 2: Criação do modelo  

:arrow_forward: Semana 3 e 4: Deploy do modelo
